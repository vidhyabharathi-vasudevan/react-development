Properties of Math:
    1) Math.PI -> to get the PI value
    2) Math.trunc -> to shorten the long value (truncation)
    3) Math.round() -> round of 4.1 -> 4 & 4.7 -> 5
    4) Math.ceil() -> 4.1 -> 5 //upper value round off
    5) Math.floor() -> 4.9 -> 4 //lower value round off
    6) Math.pow(2,4); -> 16 -> to find the power 
    7) Math.min(1,4) -> 1 //we can pass more than 2 values as we want
    8) Math.max(90,2) -> 90 //,,
    9) Math.random() // 0.30044538876259164 - any random value between 0 - 1
